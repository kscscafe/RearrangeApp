# RearrangeApp

中国語（簡体字）の例文の並び替え問題のWebアプリです。

中国語のピンイン変換ライブラリ「pinyin」を使用しています。
https://www.npmjs.com/package/pinyin

2パターンの並び替え練習が可能です。
（１）日本語から単語を並び替える
（２）日本語訳を非表示にして、音声を聞いて、単語を並び替える
